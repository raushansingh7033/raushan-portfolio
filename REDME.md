# Raushan Singh Portfolio

This is my personal portfolio built with **React, Vite, and TailwindCSS**.  
It showcases my projects, skills, and achievements.

## Features

- Dark/Light mode support
- Smooth animations with Framer Motion
- Interactive project sections
- Typewriter effect for hero text
- Particle background effects

## Tech Stack

- React 19
- Vite
- TailwindCSS 3
- Framer Motion
- React Router
- tsparticles
- Typewriter Effect
